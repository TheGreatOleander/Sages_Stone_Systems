## Installation

Systems depends on both Core and Runtime:

```bash
# Clone all three
git clone https://github.com/TheGreatOleander/Sages_Stone.git
git clone https://github.com/TheGreatOleander/Sages_Stone_Runtime.git
git clone https://github.com/TheGreatOleander/Sages_Stone_Systems.git

# Install Core
cd Sages_Stone
pip install -e .

# Runtime has no dependencies by design
# (its presence is sufficient)

# Install Systems
cd ../Sages_Stone_Systems
pip install -e .
```

If this fails, something fundamental is wrong with your environment. That information is valuable.
